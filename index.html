FCC Go Image Search Abstraction Layer

This is a url shortener microservice written with Go as a solution to Free Code Camp's URL Shortener Microservice API Project

Instructions

  • Search for images Parameters:
    • offset: int default 1,
    {{ .Host }}/api/imagesearch/lolcats?offset=1 Expected output: 
    [
    {
        url: "http://i0.kym-cdn.com/photos/images/original/001/284/242/202.jpg",
        snippet: "LOLcats | Know Your Meme",
        thumbnail: "https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcRwJE7uvsmEKEj2i-GEWuOJ_pBy8OTx2guE3bFWFkINcl4tmMZ37OBPn7Hb",
        context: "http://knowyourmeme.com/photos/1284242-lolcats"
    }
]
